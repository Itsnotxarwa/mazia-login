# Login Page

This project is a login interface built with React and Vite. It is part of a system that includes separate dashboards for users and administrators.

## Purpose

The login page authenticates users and redirects them based on their role:

* **User → User Dashboard**
* **Admin → Admin Dashboard**

### Installation

1. Clone the repository

git clone https://github.com/your-username/login-page.git

2. Navigate to the project folder

cd login-page

3. Install dependencies:

npm install

4. Start development server:

npm run dev

The application will run at:
http://localhost:5173
